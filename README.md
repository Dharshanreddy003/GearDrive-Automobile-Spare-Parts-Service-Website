# GearDrive - Automobile Spare Parts & Service Website 🚗⚙️

GearDrive is a modern web application that provides a one-stop solution for automobile spare parts and services. Users can browse spare parts, book services like car washing, painting, and modification, and easily manage their cart.

## 🌐 Website Pages

- **Home Page** – Overview of services and navigation to different sections.
- **Spare Parts Page** – Display various automobile spare parts with filters by vehicle brand.
- **Painting Page** – Service booking for custom painting and dent repair.
- **Washing Page** – Car/bike cleaning and detailing service booking.
- **Modification Page** – Customize vehicle appearance and performance.
- **Cart Page** – Add and manage spare parts or services.
- **Login & Register Pages** – Secure authentication system for users.

## 🛠️ Technologies Used

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Version Control:** Git & GitHub

## ⚙️ How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/Dharshanreddy003/GearDrive-Automobile-Spare-Parts-Service-Website.git
   ```
2. Open the project in VS Code or any editor.
3. Set up a local server (XAMPP/WAMP/LAMP).
4. Place the project in the server directory (e.g., `htdocs` for XAMPP).
5. Import the MySQL database (if available).
6. Access the site at: `http://localhost/GearDrive/index.html`
